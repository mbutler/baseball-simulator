GameState

GameState

Type: Object

Properties

top (boolean) : True if top half, false if bottom
outs (number) : Number of outs in current half-inning

pitcherFatigue (Array<{battersFaced: number}>) : Array tracking batters faced for each pitcher ([away, home])
